<p>The OGC certification program gives a third party validation that the a web service is compliant with the standard.</p>
<p>The certification process requires manual work, so we will only certify on version for each LTR.  This was not enough, so we build a <a href="http://test.qgis.org/ogc_cite/">OGC CI test platform</a> that is checking compliance <strong>every night</strong> for WMS and WFS, so that you can check by yourself any specific version commit.</p>
<p>However WMS 1.3 is only the basics, if you rely on other services like WFS, WCS, or advanced capabilities like <a href="http://cite.opengeospatial.org/teamengine/about/wms/1.3.0/site/">Raster or Vector Elevation</a>, we are looking for supporters!</p>
<p>As the future is almost now If you want QGIS to be on the cutting edge with the upcoming WFS3, a JSON-REST modern version of WFS, please get in touch. We&#8217;d love to push this both into QGIS server and Desktop.</p>
