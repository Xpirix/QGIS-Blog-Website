<p class="wp-block-paragraph">The wait is over! We are pleased to announce the new major release of QGIS 4.0.</p>



<p class="wp-block-paragraph"><a href="https://qgis.org/en/site/forusers/download.html">Installers for</a> Windows, Linux, and Mac are already out.</p>

<h2 class="wp-block-heading">What&#8217;s new?</h2>



<p class="wp-block-paragraph">On the surface, existing users should expect to engage with a QGIS experience familiar to what they have come to know and love from previous releases. Under the hood, however, 4.0 introduces significant changes to maintainability and usability. These changes ensure that QGIS 4.0 can unlock additional access to modern libraries while bringing much-needed performance and security improvements to the code base.</p>



<h3 class="wp-block-heading">For developers</h3>



<p class="wp-block-paragraph">To ensure a smooth transition, we have retained deprecated APIs where possible, minimising the effort required for plugin developers to update their tools. While some legacy APIs (such as the Processing API from QGIS 2.x) will not be guaranteed future support and backwards compatibility throughout the lifespan of the QGIS 4.x series, developers supporting existing plugins can easily ensure their plugins are compatible with the new release using the&nbsp;<a href="https://github.com/qgis/QGIS/wiki/Plugin-migration-to-be-compatible-with-Qt5-and-Qt6" target="_blank" rel="noreferrer noopener">Qt6 compatibility guide</a>.</p>



<h3 class="wp-block-heading">New features</h3>



<p class="wp-block-paragraph">While preparation for the QGIS 4.0 migration has been underway, the developer community has added over 100 new features across the application, making QGIS more powerful, more flexible, more secure, and generally just more awesome. Adjacent to developments associated with the code base, the budding community of QGIS users has continued to share resources, including projects, styles, scripts, and more, leading to an exciting period of growth for the revamped&nbsp;<a href="https://hub.qgis.org/" target="_blank" rel="noreferrer noopener">QGIS Hub</a>&nbsp;and associated community sites.</p>



<p class="wp-block-paragraph">For a whirlwind tour of all the new functionalities introduced in this release, you can view the&nbsp;<a href="https://youtu.be/wTK1exokBA8" target="_blank" rel="noreferrer noopener">highlight reel video on YouTube</a>, and for a detailed rundown of the new features and improvements, please check out the&nbsp;<a href="https://qgis.org/project/visual-changelogs/visualchangelog40/">visual changelog</a>&nbsp;for this release.</p>
