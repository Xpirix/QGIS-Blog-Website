<p>We are very pleased to announce the release of QGIS 2.10 &#8216;Pisa&#8217;. Pisa was the host city to our developer meet up in March 2010.</p>
<h1>Latest Release</h1>
<p>This is another release following our four monthly schedule.  It again brings many nice new features to QGIS.  With the release of 2.10 the previous release 2.8 &#8216;Wien&#8217;, which is designated a long term release (LTR) is moved to the long term package repositories and is the first to arrive there.  If you are working in a production environment where you wish to be more conservative about rolling out new features to your users, you will probably prefer those. Of course going with the feature frozen LTR also means that you&#8217;ll have to learn<br />
to do without all the nice new things introduced in 2.10 and above until the next long term is released next year.</p>
<h1>New Features in QGIS 2.10 &#8216;Pisa&#8217;</h1>
<p>QGIS 2.10 includes many great new features, tweaks and enhancements to make the most popular Free desktop GIS even more feature filled and useful.  Visit the visual changelog that highlights some of the new additions (<a href="http://qgis.org/en/site/forusers/visualchangelog210/index.html" target="_blank" rel="noreferrer">http://qgis.org/en/site/forusers/visualchangelog210/index.html</a>).</p>
<p>Whenever new features are added to software they introduce the possibility of new bugs &#8211; if you encounter any problems with this release, please file a ticket on the QGIS Bug Tracker (<a href="http://hub.qgis.org/" target="_blank" rel="noreferrer">http://hub.qgis.org</a>).</p>
<p>The source code and binaries for Windows, Debian and Ubuntu are already available via the large download link on our home page: <a href="http://qgis.org/" target="_blank" rel="noreferrer">http://qgis.org</a>.  More packages will follow as soon as the package maintainers finish their work. Please revisit the page if your platform is not available yet.</p>
