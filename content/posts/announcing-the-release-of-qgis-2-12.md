This week we have more great news from Jürgen Fischer, our release manager:</p>
<p>&#8220;QGIS is a user friendly Open Source Geographic Information System that runs on Linux, Unix, Mac OSX, and Windows.</p>
<p>We are very pleased to announce the release of QGIS 2.12 &#8216;Lyon&#8217;. Lyon was the host city to our developer meet up in April 2012.</p>
<p><strong>Latest Release</strong></p>
<p>This is another regular release following our four monthly schedule.  It again brings many nice new features to QGIS.</p>
<p>Please also note that we&#8217;re constantly keeping our long term release QGIS 2.8 &#8216;Wien&#8217; in good shape to better suit more conservative production setups.</p>
<p><strong>New Features in QGIS 2.12 &#8216;Lyon&#8217;</strong></p>
<p>QGIS 2.12 includes many great new features, tweaks and enhancements to make the<br />
most popular Free desktop GIS even more feature filled and useful.  Visit the <a href="http://qgis.org/en/site/forusers/visualchangelog212/">visual changelog</a> that highlights some of the new additions.</p>
<p>Whenever new features are added to software they introduce the possibility of new bugs &#8211; if you encounter any problems with this release, please file a ticket on the QGIS Bug Tracker (<a href="http://hub.qgis.org/">http://hub.qgis.org</a>).</p>
<p>The source code and binaries for Windows, Debian and Ubuntu are already available via the large download link on our home page: <a href="http://qgis.org/">http://qgis.org</a>.  More packages will follow as soon as the package maintainers finish their work. Please revisit the page if your platform is not available yet.&#8221;</p>
